## 🌐 Enable GitHub Pages (One-Time Setup)

### Step 1: Go to Repository Settings
1. Visit: https://github.com/Ashoka36/Shiva-s-Reset-Retreat
2. Click **Settings** (gear icon, top right)

### Step 2: Navigate to Pages
1. In left sidebar, scroll down to **Pages**
2. Click on **Pages**

### Step 3: Configure Pages Source
1. Under "Source", select **Deploy from a branch**
2. Select branch: **master**
3. Select folder: **/ (root)**
4. Click **Save**

### Step 4: Wait for Deployment
- GitHub will deploy your site in 1-2 minutes
- You'll see a green checkmark when complete
- A URL will appear: `https://ashoka36.github.io/Shiva-s-Reset-Retreat/`
